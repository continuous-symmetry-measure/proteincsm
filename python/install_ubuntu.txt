Ubuntu installations for CSM

1. Python 3.5.1 (http://askubuntu.com/a/682875/259462):
	sudo add-apt-repository ppa:fkrull/deadsnakes
	sudo apt-get update
	sudo apt-get install python3.5
2. Python 3.5.1 Dev:
	sudo apt-get install python3.5-dev
3. g++
	sudo apt-get install g++
4. OpenBabel
	sudo apt-get install openbabel libopenbabel-dev
4. Python Virtualenv 
	sudo apt-get install python-virtualenv
5. Install SWIG
	sudo apt-get install swig
6. Install Eigen
	sudo apt-get install libeigen3-dev
	
To set up the virtual environment:

1. Create a file called ~/.pydistutils.cfg and copy the following content:
[build_ext]
include_dirs=/usr/include/openbabel-2.0;/usr/include/eigen3
library_dirs=/usr/lib

2. pip install numpy
3. pip install openbabel==1.8.4
4. pip install csm --extra-index-url https://repo.fury.io/zmbq/
