CSM Installation on CentOS

1. Install g++
	sudo yum install gcc-c++
2. Install Python 3.5.1
	Python 3.5.1 needs to be compiled on CentOS (explanations from here http://idroot.net/tutorials/how-to-install-python-3-on-centos-7/):
	1. Install all the development tools
		sudo yum groupinstall "Development tools"
	2. Download the Python sources (you may need to install wget)
		wget https://www.python.org/ftp/python/3.5.1/Python-3.5.1.tgz
	3. Unzip:
		tar xzf Python-3.5.1.tgz
		cd Python-3.5.1
	4. Install openssl-devel for SSL/TLS (required by pip)
		sudo yum install openssl-devel
	5. Configure
		./configure --prefix=/usr/local
	6. Install
		sudo make altinstall
3. Add the EPEL repository for openbabel (https://support.rackspace.com/how-to/install-epel-and-additional-repositories-on-centos-and-red-hat/):
	sudo yum install epel-release
4. Install openbabel
	sudo yum install openbabel-devel
5. Install SWIG
	sudo yum install swig
6. Download the Eigen header files and place them in a nice place:

	cp -r eigen-eigen-07105f7124f9/Eigen ~/include
